# 帮助

>探索web极限，收集优雅、炫酷、好用的web应用。

## 快捷键

- 空格键：显示/隐藏应用面板
- ctrl键+/键：显示/隐藏搜索面板
- ctrl键+u键：清空搜索词并聚焦搜索框
- 滚轮/上下键：切换搜索引擎
